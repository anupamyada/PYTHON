SETS IS DATA STRUCTURE USED TO STORE MULTIPLE OBJECTS WITH SINGLE NAME
SETS IS MUTABLE
SETS DOES NOT ALLOW DUPLICATE ELEMENT
SET ALLOW HETEROGENEOUS ELEMENT
SETS DOES NOT SUPPORTS INDEXING AND SLICING
IT IS USED TO PERFORM MATHEMATICAL OPERATION SUCH AA UNION INTERSECTION
IT ALLOWS TO STORE ONLY IMMUTABLE ELEMENTS
SETS MODIFICATION POSSIBLE ONLY BY PERFORMING UNION AND INTERSECTION
